# Release scope

## Public/internal GitHub archive

Include software-only materials:

- wrappers and scripts for ChimeraX, ISOLDE, PHENIX, Coot, Servalcat/REFMAC5 and gemmi workflows;
- sanitized skill protocols;
- architecture documentation;
- environment/version documentation;
- privacy/redaction policy.

Exclude:

- unpublished coordinates/maps;
- private OpenClaw memory, routing configuration, tokens, identity files, chat configs;
- raw session logs;
- reviewer-only case-study evidence for Examples 1, 2 and 3.

## Reviewer-only evidence bundle

Examples 1, 2 and 3 are to be provided only for reviewer/editor checking. Example 2 target identity/PDB/accession information must remain withheld publicly and provided only if reviewers request it.
