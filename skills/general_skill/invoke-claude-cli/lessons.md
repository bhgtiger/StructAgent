## Distilled 2026-05-20

- Claude CLI may stall or get stuck when asked to directly edit a large/complex markdown artifact. Workaround: give Claude a compact source pack and ask it to produce the target markdown as a draft, then write/review the file manually in the orchestrator session before marking done.
