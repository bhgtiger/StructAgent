# RELION 5 CLI capture

Sanitized command-help capture from a RELION 5.0.0 installation used while building this skill.

- `help/` contains `relion_* --help` / `-h` output for the programs needed by the skill.
- `env/program_availability.txt` and `env/relion_binaries_on_path.txt` are sanitized inventories; paths are generalized to `<RELION_BIN>`.

For a live site, prefer `relion_<program> --help` on that machine over this bundled capture.
