# StructAgent

StructAgent is a user-guided multi-agent workflow system for cryo-EM model building and refinement. It coordinates established structural biology tools through auditable protocols while keeping scientific decision authority with the user.

## Release status

This is an internal software-archive skeleton for review before GitHub publication under `bhgtiger`.

Public release scope is currently limited to software/protocol materials only:

- tool wrappers and scripts;
- sanitized skill/protocol documents;
- architecture and environment documentation;
- privacy/redaction guidance.

The case-study examples, structural data, derived models, and detailed session logs for Examples 1–3 are reviewer-only and are not included in this public software archive.
